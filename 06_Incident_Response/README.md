# Incident Response

Incident Response (IR) is a core function of Blue Team and SOC operations. It focuses on **detecting, containing, and recovering from security incidents** while minimizing business impact.

This module presents Incident Response from a **practical SOC perspective**, aligned with BTL1 and real-world operational environments.

The main objectives of Incident Response are:

- Restore normal operations
- Document findings and lessons learned

Incident Response is both a **technical and organizational process**.
