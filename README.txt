BTL1 STUDY GUIDE – BLUE TEAM LEVEL 1

This repository is a practical study guide and portfolio focused on
defensive cybersecurity and SOC operations.

Purpose:
- Prepare for the BTL1 certification
- Practice real-world Blue Team workflows
- Demonstrate SOC analyst skills

Modules:
- Introduction to BTL1
- Phishing Analysis
- Threat Intelligence
- Digital Forensics
- SIEM Analysis
- Network Analysis
- Incident Response

Focus:
- Detection and response
- Evidence-based investigation
- Log and traffic analysis
- Incident handling and documentation

This project follows official frameworks such as NIST, MITRE ATT&CK,
CISA and Security Blue Team guidelines.
