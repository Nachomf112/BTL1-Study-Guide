# Network Analysis

Network analysis is a core skill for Blue Team and SOC analysts. By inspecting network traffic, analysts can identify malicious behavior, validate alerts, and reconstruct attacker activity.

This module focuses on **practical network analysis for detection and investigation**, not network engineering or offensive exploitation.

The main objectives of network analysis are:

- Support incident investigation and scoping
- Provide evidence for escalation or containment
